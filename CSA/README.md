# Computer Science A (CSA) – VS Code Setup Guide

Welcome to Computer Science A! This course uses **Java**, so we need to set up VS Code with the right Java tools. Follow every step carefully.

---

## ✅ Step 1: Install VS Code

If you haven't already, download and install VS Code:

👉 [https://code.visualstudio.com/](https://code.visualstudio.com/)

Run the installer and follow the on-screen instructions. Accept all default settings.

---

## ✅ Step 2: Install Required Extensions

Open VS Code, then click the **Extensions** icon in the left sidebar, or press `Ctrl+Shift+X`.

Search for and install the following extensions:

### ☕ Course-Specific Extensions (Java)

#### 1. Extension Pack for Java
A collection of popular Java tools for VS Code — includes everything you need to write, run, and debug Java programs.

- **Publisher:** Microsoft  
- 🔗 [Install Extension Pack for Java](https://marketplace.visualstudio.com/items?itemName=vscjava.vscode-java-pack)
- Search: `Extension Pack for Java` by Microsoft

#### 2. Java (Oracle)
Provides core Java language support powered by Oracle.

- **Publisher:** Oracle  
- 🔗 [Install Java by Oracle](https://marketplace.visualstudio.com/items?itemName=Oracle.oracle-java)
- Search: `Java` by Oracle

#### 3. Language Support for Java™ by Red Hat
Adds rich Java language features including IntelliSense, code navigation, and refactoring.

- **Publisher:** Red Hat  
- 🔗 [Install Language Support for Java™ by Red Hat](https://marketplace.visualstudio.com/items?itemName=redhat.java)
- Search: `Language Support for Java` by Red Hat

---

### 🌐 Extensions Required for ALL Classes

Install these three extensions as well — every student in the department needs them:

#### 4. Live Share
Collaborate on code with your teacher or classmates in real time.

- **Publisher:** Microsoft  
- 🔗 [Install Live Share](https://marketplace.visualstudio.com/items?itemName=MS-vsliveshare.vsliveshare)
- Search: `Live Share` by Microsoft

#### 5. Office Viewer
View and open Word documents, Excel spreadsheets, and PDFs directly inside VS Code.

- **Publisher:** cweijan  
- 🔗 [Install Office Viewer](https://marketplace.visualstudio.com/items?itemName=cweijan.vscode-office)
- Search: `Office Viewer` by cweijan

#### 6. WSL
Connects VS Code to Windows Subsystem for Linux (WSL).

- **Publisher:** Microsoft  
- 🔗 [Install WSL](https://marketplace.visualstudio.com/items?itemName=ms-vscode-remote.remote-wsl)
- Search: `WSL` by Microsoft

---

## ✅ Step 3: Verify Your Installation

Once all extensions are installed, check that the following appear in your Extensions panel under **Installed**:

- [ ] Extension Pack for Java
- [ ] Java (Oracle)
- [ ] Language Support for Java™ by Red Hat
- [ ] Live Share
- [ ] Office Viewer
- [ ] WSL
