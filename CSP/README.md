# Computer Science Principles (CSP) – VS Code Setup Guide

Welcome to Computer Science Principles! This course uses **Python** as its primary language. Follow every step below to get VS Code ready.

---

## ✅ Step 1: Install VS Code

If you haven't already, download and install VS Code:

👉 [https://code.visualstudio.com/](https://code.visualstudio.com/)

Run the installer and follow the on-screen instructions. Accept all default settings.

---

## ✅ Step 2: Install Required Extensions

Open VS Code, then click the **Extensions** icon in the left sidebar, or press `Ctrl+Shift+X`.

Search for and install the following extensions:

### 🐍 Course-Specific Extensions (Python)

#### 1. Python
Core Python language support for VS Code — enables running and editing Python scripts.

- **Publisher:** Microsoft  
- 🔗 [Install Python](https://marketplace.visualstudio.com/items?itemName=ms-python.python)
- Search: `Python` by Microsoft

#### 2. Python Debugger
A dedicated debugger for Python code — helps you find and fix errors step by step.

- **Publisher:** Microsoft  
- 🔗 [Install Python Debugger](https://marketplace.visualstudio.com/items?itemName=ms-python.debugpy)
- Search: `Python Debugger` by Microsoft

#### 3. Python Extension Pack
A bundle of popular Python tools including Pylance, Jupyter, and more.

- **Publisher:** Don Jayamanne  
- 🔗 [Install Python Extension Pack](https://marketplace.visualstudio.com/items?itemName=donjayamanne.python-extension-pack)
- Search: `Python Extension Pack` by Don Jayamanne

---

### 🌐 Extensions Required for ALL Classes

Install these extensions as well — every student in the department needs them:

#### 4. Live Share
Collaborate on code with your teacher or classmates in real time.

- **Publisher:** Microsoft  
- 🔗 [Install Live Share](https://marketplace.visualstudio.com/items?itemName=MS-vsliveshare.vsliveshare)
- Search: `Live Share` by Microsoft

#### 5. Office Viewer
View and open Word documents, Excel spreadsheets, and PDFs directly inside VS Code.

- **Publisher:** cweijan  
- 🔗 [Install Office Viewer](https://marketplace.visualstudio.com/items?itemName=cweijan.vscode-office)
- Search: `Office Viewer` by cweijan

#### 6. WSL
Connects VS Code to Windows Subsystem for Linux (WSL).

- **Publisher:** Microsoft  
- 🔗 [Install WSL](https://marketplace.visualstudio.com/items?itemName=ms-vscode-remote.remote-wsl)
- Search: `WSL` by Microsoft

---

## ✅ Step 3: Verify Your Installation

Once all extensions are installed, check that the following appear in your Extensions panel under **Installed**:

- [ ] Python
- [ ] Python Debugger
- [ ] Python Extension Pack
- [ ] Live Share
- [ ] Office Viewer
- [ ] WSL
