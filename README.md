## 🔧 Extensions Required for ALL Classes (except C#)

Regardless of your course, every VS Code student (except C# students) must have these three extensions installed:

| Extension | Link |
|---|---|
| **Live Share** – Collaborate on code in real time | [Install](https://marketplace.visualstudio.com/items?itemName=MS-vsliveshare.vsliveshare) |
| **Office Viewer** – View Word, Excel, PDF files inside VS Code | [Install](https://marketplace.visualstudio.com/items?itemName=cweijan.vscode-office) |
| **WSL** – Connect VS Code to Windows Subsystem for Linux | [Install](https://marketplace.visualstudio.com/items?itemName=ms-vscode-remote.remote-wsl) |

## 💻 What is VS Code?

Visual Studio Code (VS Code) is a free, lightweight, and powerful code editor made by Microsoft. It supports virtually every programming language and can be extended with **extensions** — add-ons that give VS Code extra features like live previews, debuggers, and language tools.

👉 [Download VS Code here](https://code.visualstudio.com/)
